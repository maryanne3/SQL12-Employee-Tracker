# SQL12-Employee-Tracker

## Description
A Back-End project which allows employers  to view and manage their employees, their roles and departments!

## Technologies 

* Inquirer.js
* MySQL2
* Node.js
* JavaScript
* Console.Table 

## Installation 
```
npm init
npm i 
npm install inquirer 
npm install console.table --save 
```
